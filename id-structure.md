# Format

File are extracted and organized following the game's format.  
The structure used is simple: `00 0000 000`
- 2 digit *Group type*
- 4 digit *ID*
- 3 digit *Index*

# Group IDs

| ID | Desc |
|---|---|
| 00 | System dialogue (tutorials, etc) |
| 04 | Uma stories (story mode) |
| 50 | Uma events (training) |
